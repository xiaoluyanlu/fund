# fund
验收:  
  ---1.克隆代码到idea(阿里云的mysqlRDS已配置)  
  ---2.启动 companies\config\CompaniesApplication.java  
  ---2.在fund文件下找到http文件,里面写好了访问接口,点击即可查验  


  batch模块(简易定时器)  
  common模块(公共使用的东西)  
  companies模块(作业里面的后端代码)  
  demo模块(读取xlsx文件,生成基础的基金数据)  
  generator模块(快速生成持久层的代码,admin,mapper)  
  http文件(测试增删改查,分页查询)  
  log文件(日志)  

版本:  
  jdk 17  
  springBoot 3.0.0
